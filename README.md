# smart-retail
Smart Retail Sales Analytics &amp; Automation
Smart Retail is a mini full-stack retail management and analytics system that helps businesses manage inventory, customers, and orders while providing sales analytics, data analytics, and business automation features.
The system includes an interactive dashboard for data visualization, inventory tracking, customer segmentation, and automated email alerts such as low stock notifications and daily sales reports.
This project was developed as a mini project to gain practical experience in full-stack development, database management, data analytics, and automation systems.
